# Packages

Typed Dart bindings for Node.js libraries.

| Package | Description |
|---------|-------------|
| `dart_node_core` | Core JS interop utilities |
| `dart_node_express` | Express.js server bindings |
| `dart_node_react` | React bindings |
| `dart_node_react_native` | React Native bindings |
| `dart_node_ws` | WebSocket bindings |
| `dart_node_better_sqlite3` | better-sqlite3 bindings |
| `dart_node_mcp` | Model Context Protocol SDK bindings |
| `dart_jsx` | JSX transpiler for Dart |
| `reflux` | Redux-style state management |
| `dart_logging` | Structured logging framework |
| `dart_node_coverage` | Code coverage for dart2js on Node.js |
